Reusing Posts from the F2015 Pelican Site

Since the previous web site was written using Pelican, it was convenient to use the Markdown metedata format. To enable this, I added markdown.extensions.meta to conf.py:

# What Markdown extensions to enable?
# You will also get gist, nikola and podcast because those are
# done in the code, hope you don't mind ;-)
# Note: most Nikola-specific extensions are done via the Nikola plugin system,
#       with the MarkdownExtension class and should not be added here.
# The default is ['fenced_code', 'codehilite']
MARKDOWN_EXTENSIONS = [
    'markdown.extensions.fenced_code',
    'markdown.extensions.codehilite',
    'markdown.extensions.extra',
    'markdown.extensions.meta'
]
